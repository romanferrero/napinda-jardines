# Ñapinda Jardines

Sitio web institucional para **Ñapinda Jardines**, una empresa uruguaya dedicada al diseño, construcción y mantenimiento de jardines.

El proyecto fue desarrollado como una **Single Page Application** utilizando React, Vite y Tailwind CSS, con foco en una experiencia visual limpia, responsive y fácil de mantener.

---

## Descripción

Ñapinda Jardines presenta sus servicios, valores, trabajos realizados y canales de contacto en un sitio moderno y adaptable a distintos dispositivos.

El contenido del sitio está organizado en secciones reutilizables y datos editables, lo que permite actualizar textos, servicios, galería o información general sin modificar directamente la estructura principal de los componentes.

---

## Tecnologías utilizadas

- **React 19** — construcción de interfaces por componentes.
- **Vite 8** — entorno de desarrollo rápido y build optimizado.
- **Tailwind CSS v4** — estilos utilitarios y sistema de diseño.
- **React Router v7** — manejo de rutas dentro de la aplicación.
- **Framer Motion** — animaciones suaves de entrada y transición.
- **ESLint** — control de calidad y consistencia del código.

---

## Funcionalidades principales

- Página principal tipo landing page.
- Sección de presentación de la empresa.
- Listado de servicios.
- Galería de trabajos realizados.
- Formulario de contacto.
- Botón flotante de WhatsApp.
- Navegación responsive.
- Animaciones sutiles al visualizar secciones.
- Página de error para rutas inexistentes.

---

## Diseño responsive

El sitio está diseñado para adaptarse correctamente a:

Celulares
Tablets
Computadoras de escritorio

Se utilizaron utilidades de Tailwind CSS para controlar espaciados, grillas, tamaños de texto y comportamiento visual según el tamaño de pantalla.
